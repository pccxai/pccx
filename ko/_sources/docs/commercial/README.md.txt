# 상용화 트랙

PCCX는 세 개의 상용화 트랙을 병렬로 운영한다. 트랙마다 대상 독자, 산출물,
계약 조건이 다르다. 트랙 간 자금 이동은 묵시적으로 연결되지 않으며,
실제 이동이 필요한 경우 적용되는 법규에 따른 별도 서면 동의가 필요하다.

## 트랙 구성

| 트랙 | 대상 | 핵심 산출물 |
| --- | --- | --- |
| [Open](open-track.md) | 연구자, 학생, 오픈소스 기여자, 평가자, 하위 OSS 사용자 | 기존 오픈 라이선스 하에서 공개된 RTL, spec, 문서, 레퍼런스 플로우 |
| [Commercial](commercial-track.md) | 지원이 필요한 고객: ProCore, Enterprise SDK, ASICKit, 모델 포팅, 통합 지원, 인증 | 별도 계약 조건의 상용화 오퍼 |
| [Capital](capital-track.md) | 스폰서, 전략 고객, 외부 투자자 | 법적 성격이 다른 자금 조달 수단 |

## 역할 분리

역할 분리의 전체 기준은
[contributor-vs-sponsor-vs-investor.md](contributor-vs-sponsor-vs-investor.md)
페이지를 단일 근거로 둔다. 이 페이지에서 contributor / sponsor /
strategic customer / investor가 각각 어떤 권리를 갖는지와 갖지 못하는지를
정의한다.

## 상태

이 디렉터리의 모든 파일은 상단의 초안 마커를 유지한다. 영문 계약,
법적 의무, 증권성 판단 등은 대상 법역에서 자격 있는 법률 자문 검토가 완료되기
전까지 구속력이 없다. 해당 검토가 끝나면 그 결과 문서를 통해 이 페이지의
효력 상태가 업데이트된다.

## 이 문서 묶음은 다음이 아니다

- 증권 인수 신청서나 투자 브로셔가 아니다.
- 계약 체결 제안이 아니다.
- 유료 협약이 요구하는 실제 계약서의 대체본이 아니다.
- contributor, 고객, 투자자에 대한 법률 자문이 아니다.
