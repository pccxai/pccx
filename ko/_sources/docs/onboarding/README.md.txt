# Onboarding

외부 리뷰어와 기여자를 위한 입구 문서다. 이 집합은 간결하게 구성되어
있으며, 권한·절차의 상세 기준은 `docs/reference/README.md`를 참조한다.

## 구성

- [Getting started](getting-started.md) — `--recurse-submodules` 클론부터
  대표 검증 실행 경로까지.
- [Architecture overview](architecture-overview.md) — 프로젝트 구조를 빠르게
  이해하기 위한 읽기 순서.
- [Contribution rules (draft)](contribution-rules.md) — 기여자가 받는 권리와
  받지 못하는 항목.

## 이 집합이 제공하지 않는 것

- 로드맵: 로드맵은 [`docs/roadmap/`](../roadmap/milestones.md)에서 관리한다.
- 아키텍처 본문: 사양 본문은 `docs/v002/`(향후 `docs/v003/`)에서 관리한다.
- 저장소 토폴로지, v002 계약, boundary rule, submodule pin policy의 최종 본문은
  `docs/reference/` 아래의 페이지에서 관리한다.
