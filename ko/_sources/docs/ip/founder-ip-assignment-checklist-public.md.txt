# Founder IP assignment checklist (public summary)

founder와 팀 멤버 기여권 설정을 위한 공개-safe 요약이다.

- [ ] 기여자(창업자 포함) 식별
- [ ] 고용 주체 기여인지 여부 확인
- [ ] 오픈 라이선스 커버 범위 검토
- [ ] 향후 entity에 대한 명시적 양도 필요 항목 식별
- [ ] work-for-hire / agency / 동의 필요성 점검

상세 체크리스트와 실제 assignment 문서는 private 저장소에서
법무 검토 후 보관한다.
