# IP 정책

PCCX의 지식재산 보호는 레이어별로 나누어 운영한다. 각 레이어는 독립된
범위와 법적 체계를 갖는다.

## Layers

- [Patent strategy](patent-strategy.md) — 후보 발명의 분류 절차
- [Trademarks](trademarks.md) — 상표명 사용 관리, 등록은 계획 단계
- [Trademark filing log](trademark-filing-log.md) — KR Class 09/42 공개 docket
- [Trade secret policy](trade-secret-policy.md) — 비공개 보안 대상을 구분
- [Contributor License Agreement](contributor-license-agreement.md) — CLA의 의도와 구조(초안)

## 운영 원칙

Open-track 산출물(PCCX 오픈 코어 RTL/spec/docs)은 기존 오픈 라이선스의 적용
범위로 처리한다. trade secret, 계약 조항, patent 카테고리는 공개 레이어와
중복되지 않는다.

## 상태

이 디렉터리의 파일은 모두 DRAFT 표시를 유지한다. 실제 등록·계약·선포는
별도 법적 문서에서 확정한다.
