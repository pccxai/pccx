# Trademarks

프로젝트는 다음 mark의 사용을 common-law 관점에서 주장한다.
공식 등록은 아직 확정되지 않았으므로, 현재 목록은 "사용 주장"임을 의미한다.

공식 진입점 및 최신 filing은
`TRADEMARKS.md`와
[`trademark-filing-log.md`](trademark-filing-log.md)를 함께 확인한다.
공개 문서에서 첫 언급에는 `PCCX™`를 사용하고,
해당 관할권 등록이 완료되기 전에는 `PCCX®`를 쓰지 않는다.

## Claim된 mark

| Mark | Use |
| --- | --- |
| **PCCX** | 프로젝트 이름, 아키텍처명, `pccxai/` 하위 저장소 family |
| **PCCX OpenCore** | 공개 코어 패키지 및 reference flow의 표시명 |
| **PCCX ProCore** | 상용 강화 RTL bundle의 표시명 |
| **PCCX ASICKit** | 상용 ASIC tape-out 대상 산출물의 표시명 |
| **PCCX Compatible** | 상업 계약에서 정의한 인증 조건을 통과한 통합의 명칭 |

## Common-law claim

상기 mark는 상표 사용이 실제 commerce에서 반복되고 있으므로,
relevant 관할권 기준으로 common-law 권리를 주장한다.
`TM` 표시는 비공식적 사용이 가능하나,
`®`는 해당 관할권 등록 후에만 사용한다.

## 등록 계획 (TBD)

공식 등록은 우선순위·관할권·class 및 선제/후속 전략이 확정되면 진행한다.

- 관할권 우선: KR 우선 + 주요 타깃 시장
- class: 전자부품, 컴퓨팅 하드웨어, 소프트웨어, 관련 서비스
- mark 우선순위: 핵심(PCCX), 제품군(PCCX ProCore, PCCX ASICKit), optional 방어 출원

## 사용 가이드

내부/외부 사용 시 가이드:
- mark를 명사 형태로 일관되게 사용 (`PCCX architecture`처럼 복수화 금지).
- 등록된 관할권에서만 `®` 사용.
- 제3자 사용은 공개된 trademark 정책(향후 공개 시)을 따름.

## 상태

현재 등록된 PCCX 상표는 없다. 위 목록은 등록된 집합이 아닌 common-law 주장이다.
