# Customer evidence pack (템플릿)

고객에게 제공되는 서면 교류용 evidence 템플릿이다. **고객 전용 상세 내용은
공개 저장소에 올리지 않는다.**

구성 항목:

- 목적/범위
- 대상 하드웨어(보드, 모델)
- 측정 지표(tokens-per-second, FPS, mAP, latency, power)는 **측정이 있을 때만** 기재
- CI/sim/formal 증거 URL
- source manifest / SHA pin
- 제한사항과 알려진 리스크(생산성 주장 제외)

측정이 없는 항목은 **TBD**로 표시하고 추정값을 채우지 않는다.
