# PCCX trademark filing log

`PCCX™` 상표 공개 상태를 위한 최소 로그이다.
private filing 자료는 의도적으로 제외한다.

## 공개 항목

- mark
- 관할권
- class
- application number
- filing date
- pending 상태
- foreign filing safety window

## 공개하지 않는 항목

- 고객번호
- 원본 XML filing
- 영수증
- 신청자 연락처
- 개인정보를 유추할 수 있는 스크린샷

개인정보 및 원본 filing은 별도 confidentiality control에서 관리되며 본 저장소에는
요약 데이터만 남긴다.

대상 관할권과 비용은 TBD.

## 사용 가이드

- 외부 공개 문서에서 첫 등장 시 `PCCX™` 사용
- 관할권별 등록 완료 전에는 `PCCX®`, `registered trademark`, `등록상표` 금지
- `PCCX Compatible`, `PCCX Certified`는 현재 계획 용어이며, 현재 공식 인증 뱃지로 쓰지 않는다.
